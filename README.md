# Comparative Evaluation of Graph Neural Network Models over Node Regression and Link Prediction Tasks

## Project Overview
This repository contains the code, experiments, and evaluation notebooks for a comprehensive study on Graph Neural Networks (GNNs). The project explores the structural and semantic capabilities of various graph architectures across multiple domains, evaluating their performance, scalability, and geometric limitations. 

The evaluation spans two primary graph-learning objectives—**Link Prediction** (Knowledge Graph Completion) and **Node Regression**—applied across three distinct, real-world datasets.

## Evaluated Tasks & Datasets

### 1. Tail Node Prediction (FB15k-237)
* **Task:** Knowledge Graph Completion (Link Prediction).
* **Objective:** Given a head entity and a directed relation, rank the most mathematically plausible tail entities.

### 2. Node Regression (NYC Taxi Dataset)
* **Task:** Spatio temporal Node Regression.
* **Objective:** Predict future demand at each node

### 3. Node Regression (Wikipedia Network)
* **Task:** Semantic & Structural Node Regression.
* **Objective:** Predict the continuous page-view count of each Wikipedia pages
